# Sakshi-E-Commerce-Sales-Dashboard
This Dashboard is  created to analyze Sakshi e-commerce sales data across multiple states, including Texas, New York, Illinois, Florida, and California. This project was a deep dive into understanding sales performance, customer behavior, and profitability trends.
The dashboard provides a comprehensive view of e-commerce sales data, including:
✅Sales Performance: Total sales, profit, and average order value (AOV) by state and quarter.
✅Customer Insights: Sales distribution by customer name and payment mode.
✅Product Trends: Quantity sold by category and subcategory, highlighting top-performing products.
✅Profit Analysis: Monthly profit trends and state-wise profitability.
